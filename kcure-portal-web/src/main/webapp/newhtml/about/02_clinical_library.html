권한별메뉴관리

  • 데이터 소개
K-CURE 라이브러리 임상 라이브러리 공공 라이브러리 질환별 라이브러리 보건복지 라이브러리
개요 데이터모델

임상 라이브러리란?

  • ・ 참여 의료기관 EMR(전자의무기록)과 CDw(임상데이터베이스)에 저장된 10개 암 임상 데이터를 암종별로 항목 표준화 진행
  • ・ 표준화된 암종별 임상 데이터를 병원별 전용 클라우드로 적재하여 임상라이브러리 구축

임상라이브러리 구축 단계

  • 암종별
    항목 표준화
  • 참여의료기관으로부터
    표준화된 데이터 수집
  • 표준화된 데이터
    클라우드에 적재

구축 계획 세부 내용

1. 암 임상 라이브러리 데이터 수집 계획

의료 데이터 중심병원 대상으로 10개암 등록 환자수 (10~23년) 165만명에 대해 임상 데이터 수집

14개 기관 참여

  • ・ 서울대병원
  • ・ 서울아산병원
  • ・ 가천대길병원
  • ・ 세브란스병원
  • ・ 아주대병원
  • ・ 화순전남대병원
  • ・ 건양대병원
  • ・ 대구가볼릭대병원
  • ・ 전북대병원
  • ・ 분당서울대병원
  • ・ 삼성서울병원
  • ・ 한림대성심병원
  • ・ 부산대병원

10개암

  • 1차년도 (22) ・ Baseline DB, 위암, 대장암
  • 2차년도 (23) ・ 유방암, 간암, 췌암
  • 3차년도 (24) ・ 폐암, 전립선암, 자궁경부
  • 4차년도 (25) ・ 신장암, 혈액암

165만명

  • ・'02 ~ 19년 암등록환자 수 기준으로 구축
  • ・ 10 ~ 23년 41개 병원 10개 암종 합계

2. 표준화 방안

1) 데이터 표준화
  • ・ 참여 의료기관 EMR(전자의무기록)과 CDw(임상데이터베이스)의 데이터를 진단, 검사, 약품 등 임상 영역별로 구분
  • ・ 주요 10개 항목 정의서 개발 및 암 임상 학회 검증 ( 추후 항목 확대 및 순차적 고도화 )
  • ・ 암 임상 라이브러리의 메타데이터를 지속적으로 관리할 수 있는 데이터 속성(Data Element)과 값(Value) 관리
구분 모집단 내용
Baseline DB 모집단 전체 (암등록 환자) 환자 기본, 진단정보, 수술정보, 투약정보 등 기본 진단 및 처치 정보
암임상
라이브러리		 공통항목 10개 암종
  • ・ 환자가족력, 신체계측치
  • ・ 진단전이, 재활
  • ・ 검사진단, 영상
  • ・ 병리병리생검, 병리외과전이, 재활
  • ・ 수술수술정보, 합병증
  • ・ 치료항암제, 방사선 등
특화항목 각 암종
  • ・ 췌장암시술
  • ・ 전립선암직장수지검사
  • ・ 신장암투석
  • ・ 자궁경부암세포병리
  • ・ 혈액암골수검사, 이식치료 등
2) 표준화 관리
  • ・ 임상용어 실무자 의견 조회 및 임상학회 검증 후, 표준화위원회를 통해 용어표준 확정
  • ・ 표준참조용어세트, KCD8 등 용어체계 교차맵을 지원하는 국제용어표준(SNOMED-CT) 한국판 표준용어서버
  • ・ EMR 원천데이터를 표준화된 상태로 수집하도록 EMR 입력창 개선 및 EMR 인증수가 시범사업 연구 반영
  • ・ 의료기관 로컬용어를 입력 시 SNOMED-CT Concept ID 및 Description ID 자동 매핑 둘 개발

데이터 적재 및 개방방안

  • ・ 기관별 연계용 Baseline DB를 병원별 관리권한이 부여된 중앙 Private Cloud의 암 임상 라이브러리 Zone에 적재
  • ・ 외부 전문가를 포함한 (가칭)K-CURE 전문위원회에서 활용•가명처리 적정성 검토, IRB 심사 후 의료데이터 안심활용센터에서 제공
