임상 라이브러리 데이터 모델

  • ・ KCURE 임상 라이브러리 데이터 모델은 기본적 임상 데이터(Baseline DB)와 특정 질환 임상 데이터(암 임상 라이브러리)에 대한 모델을 검색/생성할 수 있도록 설계되었습니다.
  • ・ 이 사이트는 데이터 모델의 현재 버전 모델 구조와 각 DB의 테이블, 항목 에 대한 내용을 제공합니다. 데이터베이스에 대한 상세 설명은 [임상라이브러리 데이터 모델]에서 확인하실 수 있습니다

임상 라이브러리 데이터 모델 v1.0 ( 현재버전)

  • ・ KCURE 임상 라이브러리 데이터 모델 V1.0 은 암 임상 라이브러리, Baseline DB, 중앙 암등록DB로 구성됩니다. 본 문서는 각 DB별 테이 블, 테이블 내 항목, 항목에서 수집되는 값 및 수집 절차 등에 대한 설명입니다.
